
# Photo Album Organizer

Vite + バニラHTML/CSS/JS + SQLite (sql.js) によるローカル写真アルバム管理アプリ

## セットアップ

```sh
npm install
npm run dev
```

## 主な特徴
- 写真を日付ごとに自動グループ化
- ドラッグ&ドロップでアルバム並べ替え
- タイル状UIで写真プレビュー
- メタデータはローカルSQLite (sql.js) に保存
- 画像ファイルはアップロードせずローカル参照のみ

## 開発
- `src/` ... 実装
- `tests/` ... テスト
- `public/` ... 静的ファイル
